# fastapi
fastapi tutes

A simple example of using Fast API in Python.

## Preconditions:

- Python 3
- FastAPI

## Install FastAPI
```
pip install fastapi
pip install uvicorn
```

### Run server

```
uvicorn fastapi_application:app --reload
```

## API documentation (provided by Swagger UI)

```
http://127.0.0.1:8000/docs
```
